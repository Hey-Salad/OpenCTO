OpenCTO dashboard prototype

This repository currently ships a React dashboard and static marketing pages. The app includes Launchpad, Jobs, Pricing, Billing, Compliance, and Settings views with mock adapters used for local testing.

Implemented UI Areas

What you can verify in the current build

Launchpad

Voice-first interface with playback controls and right-side configuration panel.

Jobs + Stream

Job list, stream view, and human approval card components rendered from typed mock data.

Billing + Pricing

Plan cards, billing summary, invoice table states, and Stripe config checks in frontend.

Auth + Compliance

Route/role guards, device/passkey settings skeleton, and compliance evidence panel.
